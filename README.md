# Semantic Gaussian Memory project page

This folder is a lightweight static share copy generated from `work2/progress_site`.

Open locally:

```powershell
python -m http.server 8776
```

Then visit `http://127.0.0.1:8776/`.

GitHub Pages sharing:

1. Put this folder contents at the root of a private GitHub repository or the `docs/` folder.
2. Enable GitHub Pages for that branch or folder.
3. Keep the repository private for internal colleague review unless the data-owner approves public release.

Evidence boundary:

- The video and 3D samples are local work2 evidence.
- The page does not include raw PLY, NPY, server logs, browser cache, verification screenshots or third-party demo media.
- Sanitized source paths are for traceability only and are not complete public artifact links.
